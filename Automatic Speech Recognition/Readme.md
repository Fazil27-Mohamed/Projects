# 🎤 Automatic Speech Recognition using OpenAI Whisper

An AI-powered web application that converts spoken audio into text using the **OpenAI Whisper** speech recognition model. The application provides a simple and interactive interface where users can upload audio files and receive accurate text transcriptions in real time.

---

## 📌 Overview

Automatic Speech Recognition (ASR) is a technology that enables computers to recognize human speech and convert it into written text. This project leverages the **OpenAI Whisper** model to deliver reliable speech-to-text transcription for various audio formats.

The application is designed with a user-friendly web interface that allows users to upload audio files, process them using the AI model, and display the generated transcript instantly. Additionally, the transcribed text is saved for future reference.

---

## ✨ Features

- 🎙️ AI-powered Speech-to-Text conversion
- 📁 Upload audio files through a web interface
- ⚡ Fast and accurate transcription using OpenAI Whisper
- 📄 Displays transcription instantly
- 💾 Automatically saves the generated transcript
- 🌐 Interactive and responsive user interface
- 🔍 Supports multiple audio formats
