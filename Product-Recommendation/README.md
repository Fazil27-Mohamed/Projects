# Product Recommendation System

A Machine Learning-based Product Recommendation System that recommends similar products based on product features such as category, brand, price, ratings, and popularity.

---

## Features

- Product Search
- Similar Product Recommendations
- Category-Based Filtering
- Brand-Based Recommendations
- Price-Based Recommendation
- Rating-Based Recommendation
- Interactive Web Interface

---

## Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- Joblib
- Pickle
- HTML
- CSS

---

## Dataset

The dataset contains product information including:

- Product Name
- Category
- Brand
- Price
- Discount Percentage
- Final Price
- Rating
- Review Count
- Units Sold
- Product Score
- Seller Rating

---

## Machine Learning Workflow

- Data Collection
- Data Cleaning
- Feature Engineering
- Data Scaling
- Product Similarity Calculation
- Model Training
- Product Recommendation
